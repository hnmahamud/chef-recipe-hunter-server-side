## Clone and Installation

For clone the repository:

```sh
git clone https://github.com/programming-hero-web-course-4/b7a10-chef-recipe-hunter-server-side-hnmahamud.git
```

For install dependency:

```sh
npm install
```

For run the project:

```sh
nodemon index.js
```
